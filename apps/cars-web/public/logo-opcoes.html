Numvi — 4 opções de monograma "N"

Minimalista · N em bordô + ouro. Cada opção mostra o símbolo + o nome. Escolha o número.
Opção 1 · N sólido (app icon)
Numvi
N branco-dourado sobre fundo bordô. Forte como ícone de app, lê bem pequeno.
Opção 2 · N com seta de alta (visão)
Numvi
A diagonal do N termina numa setinha bordô pra cima — "dinheiro com visão / em alta".
Opção 3 · N no anel (moeda)
Numvi·finanças
Mesmo formato do Aureus (anel) mas com N no centro. Cara "financeira" imediata.
Opção 4 · N traçado (ultra minimal)
Numvi
N feito de um traço só + ponto bordô no topo. Leve, moderno, respira.
